# Third Wave

 A web application that allows users to add, view and view a list of third wave coffee shops.

# Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Express.js
- Node.js
- MongoDB
- Mongoose
- EJS
